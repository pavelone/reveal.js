Problem

The cost of development and implementation of healthcare applications is very high

Solution

Aidbox - a platform for modern web and mobile healthcare applications built on top of a centralized FHIR server


Cuts cost through time to market improvements

Aidbox

  1. get a FHIR server in the cloud
  2. use HL7 connectors to get data from the EHR (Electronic Health Record) systems
  3. clone a sample app
  4. follow a tutorial
  5. use GUI building blocks
  6. deploy an app into a secure hosting

What can you build with Aidbox?

  • patient engagement apps

  • HIE solutions

Aidbox platform

Aidbox features

  • a scalable database and FHIR® compliant REST API
  • OAuth and flexible security rules to control access
  • secure HIPAA compliant hosting
  • SDK and tutorials for different server and mobile platforms including Java, .NET, JavaScript, Ruby, Python and iOS
  • template applications and GUI building blocks
  • connectors for integration with EHR systems and medical devices (HL7 v.2 to FHIR etc)
  • SMART on FHIR compatible apps developed by Health Samurai and other developers
